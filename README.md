# Active-Directory-Lab
A step-by-step Active Directory lab using Windows Server 2025 and Windows 11, running in VirtualBox.

## Goals
- Install and configure Windows Server in VirtualBox
- Install Windows OS Client VM in VirtualBox
- Join the client device to the domain
- Create users, groups, and OUs
- Deploy GPOs for domain management

## What do you need?
- Host machine with at least 16GB RAM (recommended), and enough disk space
- VirtualBox
- Windows Server ISO (I am using Windows Server 2025 ISO)
- Windows OS ISO (I am using Windows 11 Pro)
